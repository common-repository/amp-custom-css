== Description ==
= Add AMP Custom CSS Code To Your AMP version =

Many websites are using AMP version to make there website Lightning fast in mobile devices but many websites are looking boring as the frontend point of view and it doesn't have the option of adding the additional CSS in the AMP verion BUT with the help of this plugin you can add your own CSS which will only work in AMP version of your website
